# Avatar Demo

This project is a demo application for displaying and animating 3D avatars using VRM files. It utilizes Vite for development and Kalidokit for motion capture and rigging.

## Features

- Load and display VRM avatar models
- Motion capture integration with Kalidokit
- Built with Vite for fast development

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd new
   ```

2. Install dependencies:
   ```
   npm install
   ```

## Usage

To start the development server:
```
npm run dev
```

To build the project:
```
npm run build
```

To preview the built project:
```
npm run preview
```

## Dependencies

- [Vite](https://vitejs.dev/) - Build tool
- [Kalidokit](https://github.com/yeemachine/kalidokit) - Motion capture library

## Project Structure

- `src/` - Source code
- `public/` - Static assets including VRM files
- `index.html` - Main HTML file
- `package.json` - Project configuration
